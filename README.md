# Для запуска потрібен 
- Node js (розроблено на версії 20.3.0)
- php 8.2.7

# Команди для запуску
## Частина бекенду. таблиці і дані в неї
```composer install```
```php artisan migrate```
```php artisan db:seed```

## Частина фронтенду. Встановлення та білд
```npm install```
```npm run build```

## Можна запускати
```php artisan serve```
